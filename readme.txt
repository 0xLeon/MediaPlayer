ISTE-Media Player
ISTE-Media Player is ein einfacher Medienspieler für MP3- und WAV-Dateien.
Vollständig unterstützt wird nur die MP3-Wiedergabe. Beim ersten Start kann
man sich mit dem Benutzername admin und dem Passwort admin einloggen. Mit diesem 
Accoun können sie weitere Accounts anlegen, Accounts bearbeiten und löschen.
Nach dem ersten Login sollten die Standard-Login-Daten geändert werden. Dies kann
für den eigenen Account unter Benutzer → Account bearbeiten. Dieser Punkt ist für
jeden Benutzer verfügbar, die Gruppenzugehörigkei lässt sich jedoch nicht ändern.
Für Administratoren findet sich unter Benutzer noch der Punkt Benutzer verwalten.
Hier können andere Accounts bearbeitet werden.
Zum Abspielen von Musik müssen über den Menüpunkt Playlist der persönlichen Playlist
Songs hinzugefügt werden. Über einen Doppelklick auf die Datei in der Playlist kann
der Song gestartet werden. Die Wiedergabe lässt sich dann über die Buttons steuern.
Auch über das Auswählen einer Datei und anschließenden Klick auf Starten kann die
Wiedergabe gestartet werden.
Dateien können aus der Playlist entfernt werden, indem man sie Auswählt und dann den
Punkt Playlist → Datei entfernen verwendet.
In der Tabelle können weitere Informationen zu den aktuellen Songs angesehen werden.
Über den Punkt Hilfe → Über ISTE-Media Player können die Lizenzbedingungen und andere
Credis eingesehen werden.
